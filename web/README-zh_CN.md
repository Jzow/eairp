## 安装与使用

- 获取项目代码

```bash
git clone https://github.com/eairps/eairp.git

cd eairp
```

如果您尚未安装pnpm工具，请使用以下命令首先安装pnpm
```bash
npm install -g pnpm
```

- 安装依赖

```bash
pnpm install
```

- 运行服务
```bash
pnpm serve
```

- 构建服务
```bash
pnpm build
```
