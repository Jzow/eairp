在ERP业务的基础上，我们扩展了GPT-3.5。个人或公司可以通过我们的系统对您的模型进行微调。
您可以通过简单的描述提供完全自动化的业务表单提交操作，还可以与GPT聊天、交互和查阅信息。
您可以通过Docker进行部署，以快速启动和使用。

它是完全免费的，如果这个项目对你有帮助，请点击Star。非常感谢。
